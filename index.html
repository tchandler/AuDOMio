AuDOMio

Hear the DOM.

Download .zip Download .tar.gz View on GitHub

Dynamically generates music by parsing the structure of web sites. Audio generation done using Audiolet.js and Music.js.

During playback, whichever nodes are being used to create a sound are given a random background and border color.
